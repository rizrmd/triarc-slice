# Acceptance Checklist

- [ ] **Project Init**: Nuxt 3 project runs locally (`npm run dev`).
- [ ] **UI Layout**: Sidebar, Canvas, and Properties panel are visible and responsive.
- [ ] **Asset Loading**: Can see images from `/characters` and user uploads.
- [ ] **Drag & Drop**: Can drag an image from sidebar to canvas.
- [ ] **Positioning**: Can drag elements on canvas to move them.
- [ ] **Selection**: Clicking an element selects it and shows its properties.
- [ ] **Properties**:
  - [ ] Can change X, Y, Width, Height.
  - [ ] Can change Z-Index (bring forward/backward).
  - [ ] Can change Opacity.
- [ ] **Visual Effects**:
  - [ ] Can apply Tint color to an image.
  - [ ] Can apply Blur to an image.
- [ ] **Templates**: Can load a predefined template (e.g., "Creature Card").
- [ ] **Grid/Snap**: Elements snap to grid when enabled.
- [ ] **Export**: Clicking "Export JSON" downloads a valid JSON file matching the schema.
