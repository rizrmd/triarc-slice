## ResourceImporterLayeredTexture <- ResourceImporter

This imports a 3-dimensional texture, which can then be used in custom shaders, as a FogMaterial density map or as a GPUParticlesAttractorVectorField3D. See also ResourceImporterTexture and ResourceImporterTextureAtlas.

**Props:**
- compress/channel_pack: int = 0
- compress/hdr_compression: int = 1
- compress/high_quality: bool = false
- compress/lossy_quality: float = 0.7
- compress/mode: int = 1
- compress/rdo_quality_loss: float = 0.0
- compress/uastc_level: int = 0
- mipmaps/generate: bool = true
- mipmaps/limit: int = -1
- slices/arrangement: int = 1
