## AudioEffectReverb <- AudioEffect

Simulates the sound of acoustic environments such as rooms, concert halls, caverns, or an open spaces.

**Props:**
- damping: float = 0.5
- dry: float = 1.0
- hipass: float = 0.0
- predelay_feedback: float = 0.4
- predelay_msec: float = 150.0
- room_size: float = 0.8
- spread: float = 1.0
- wet: float = 0.5
