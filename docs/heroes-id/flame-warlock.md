# Malachar, Sang Flame Warlock

## Lore

Dahulu dia adalah sarjana arcane brilian di akademi prestisiu, Malachar terobsesi untuk mencapai batas teoritis sihir api. Dia percaya api bukan hanya elemen — itu kekuatan hidup yang ingin dibebaskan. Rekannya menyebutnya bidah. Dia menyebutnya kebenaran.

Dia merancang eksperimen untuk menyalurkan chaos-wind mentah sebagai akseleran melalui sigil api, berharap menciptakan nyala api yang berkelanjutan sendiri. Berhasil — terlalu bagus. Nyala api tidak hanya bertahan sendiri, itu menemukan inang. Dia. Api membakar pikirannya, fuses dengan pikiran-pikirannya. Dia tidak kehilangan kecerdasanannya — dia kehilangan kendalinya. Setiap pikiran sekarang diakhiri dalam api. Setiap solusi adalah membakar.

Dia tidak kehilangan akal. Itu yang membuatnya berbahaya. Dia cukup lucid untuk stratregize, cukup articulate untuk mengejek musuhnya, tapi fondamental broken — akalnya berkedip seperti lilin di angin yang memberi makan dia. Dalam momen tenang yang rare, dia ingat siapa dia. Dia membenci momen-momen itu.

Mivasinya bukan kekuatan — dia sudah punya. Dia mau keheningan. Tapi satu-satunya cara api itu diam adalah ketika segalanya di sekitarnya sudah abu.

## Kepribadian

Arogant, destructure, consumed oleh nyala api. Berbicara tentang membakar, abu, dan chaos. Melihat musuh sebagai bahan bakar. Tidak menghormati apapun kecuali kekuatan mentah. Lucid enough untuk kejam, broken enough untuk unpredictable.

## Hubungan Lore

- **Frost Queen** — Nemesis alami. Dinginnya adalah satu-satunya hal yang mendiamkan api di kepalanya. Dia menakutkannya dan membenci ketakutan itu.
- **Princess Emberheart** — Bukti bahwa api bisa dikendalikan dengan disiplin. Dia mendapati displinnya menyinggung. Elle melihatnya sebagai kisah peringatan.
- **Wind Monk** — Angin memberi makan api Malachar, tapi Biksu melihatnya sebagai corrupt dari alam sejati angin.
- **Dawn Priest** — Mungkin bisa memurnakannya. Dia menghindari cahaya.
- **Necromancer** — Roe-roh dalam kegilaan. Necromancer memahami obsesi — mereka berdagang kematian seperti Malachar berdagang api.
- **Demon Empress** — Elle menghormati kekuatan mentah. Il menghormati dominasi atas api neraka. Tidak ada yang saling percaya.
- **Spellblade Empress (Mirael)** — Elle adalah rekan sekali. Elle mencoba mengajarkan dia keseimbangan. Il bilang keseimbangan adalah pengecut. Elle menjadi headmistress. Il menjadi api. Elle adalah bukti bahwa kontrol mungkin. Il mendapati itu menyinggung.
- **Blood Alchemist (Cael)** — Api membuat kulit Cael merinding. Malachar tidak perhatikan atau peduli. Tapi target yang terbakar berdarah dengan bebas, dan Cael mengikuti seperti burung nasar追随 wildfire. Sinergi mereka kimiawi, bukan personal.
