Renzo, the Gunslinger - Chamber Grid

## Resource Utama

### 3 Chamber
- Setiap action card mengisi 1 bullet ke chamber kosong berikutnya
- Jika semua chamber penuh, pemain harus menembak atau ganti chamber
- Urutan chamber menentukan efek kombinasi

## Element Bullet Types

| Element | Efek |
|---------|------|
| 🔥 Fire | Damage lebih tinggi / ledakan kecil. Cocok untuk burst. |
| 💨 Wind | Kecepatan tembak / tembakan menembus target. Cocok untuk tempo & pressure. |
| 🪨 Earth | Impact berat, stagger, perlindungan kecil setelah tembak. Cocok untuk duel. |
| ✨ Light | Akurasi+, mark, bonus follow-up. Cocok untuk setup kill. |
| 🌑 Shadow | Execution damage, anti-heal, finishing. Cocok target lemah. |
| ❄️ Ice | Slow, pin, recoil control. Cocok untuk kontrol target. |

## Payoff Shots

| Shot | Requirement | Efek |
|------|-------------|------|
| Quick Shot | 1 chamber terisi | Tembak 1 bullet segera. Reaksi cepat tanpa setup. |
| Double Tap | 2 chamber terisi | Tembak 2 bullet berurutan (satu atau dua target). Pressure taktis. |
| Trick Combo | Kombinasi khusus | Fire+Wind = peluru meledak menyebar cepat<br>Ice+Light = tembakan penanda memudahkan follow-up<br>Earth+Shadow = shot berat menghancurkan target lemah |
| Full Magazine | 3 chamber penuh | Burst penuh, semua bullet ditembak beruntun dengan bonus tempo/crit. Payoff terbesar. |

## Keputusan Pemain

1. **Quick Shot**: Isi 1 chamber → tembak cepat
2. **Double Tap**: Simpan 2 chamber untuk combo pendek
3. **Trick Combo**: Rakit kombinasi chamber tertentu untuk efek spesial
4. **Full Magazine**: Susun 3 bullet pas → lepaskan full burst

## Inti Gameplay

Chamber Grid membuat Renzo terasa seperti marksman teknis, bukan caster biasa. Kekuatan datang dari cara memuat, menyusun, dan melepas urutan peluru elemental pada momen tepat.
