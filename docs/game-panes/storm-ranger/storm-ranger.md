Hero ini bernama Seryn, the Storm Ranger.

Konsep gameplay Seryn sederhana:

Seryn tidak memakai action card seperti hero biasa. Kalau pemain memberi action card ke Seryn, card itu tidak keluar sebagai spell asli. Sebagai gantinya, Seryn mengubah card itu menjadi skill panah miliknya sendiri. Jadi card hanya jadi pemicu.

Sistem ini bisa disebut Arrow Doctrine:

elemen card menentukan skill apa yang keluar
cost card menentukan seberapa kuat skill itu
Mapping-nya seperti ini:

Wind jadi Gale Volley
Light jadi Hunter’s Mark
Ice jadi Pinning Shot
Fire jadi Burst Arrow
Earth jadi Anchored Guard
Shadow jadi Veiled Shot
Tier kekuatannya:

cost 1-2 = versi ringan
cost 3 = versi penuh
cost 4 = versi terkuat
Jadi cara main Seryn adalah:

pilih elemen card sesuai kebutuhan
pakai Light untuk setup target
pakai Ice untuk menahan target
pakai Fire atau Shadow untuk menghabisi
