Zyara, Princess Emberheart punya game pane unik bernama Crownfire Standard.

Konsep gameplay-nya:

Zyara tidak membiarkan apinya liar seperti Malachar.
Dia membawanya sebagai simbol kerajaan: api yang diarahkan, ditahan, lalu dilepas sesuai kebutuhan pasukan.
Pane ini membuat pemain memilih `Standard` aktif, yaitu bagaimana api kerajaan Zyara sedang dipakai saat itu.

Tiga Standard utama Zyara:

Advance
api dipakai untuk mendorong serangan dan momentum

Holdfast
api dipakai untuk menahan garis dan melindungi sekutu

Judgment
api dipakai untuk menghukum satu musuh yang dianggap ancaman utama

Saat Zyara menerima action card, menyerang, atau menahan serangan di garis depan, dia membangun `Crownfire`.
Crownfire maksimum adalah 5.

Di hero pane, pemain akan melihat:

Standard mana yang sedang aktif
jumlah Crownfire saat ini
musuh mana yang sedang menjadi target Judgment jika mode itu dipilih
skill pelepasan apa yang terbuka dalam Standard sekarang

Cara main tiap Standard:

Advance
memberi bonus ke tekanan ofensif Zyara
serangan berikutnya bisa jadi lebih cepat, cleave kecil, atau memberi buff serang ke ally terdekat
ini cocok saat tim ingin menekan lebih dulu

Holdfast
memberi nilai defensif ke Crownfire
Zyara bisa membagi shield, intercept, atau pengurangan damage ke ally yang terancam
ini cocok saat tim perlu bertahan sambil tetap berada di garis depan

Judgment
pemain memilih 1 musuh sebagai target hukuman
selama Standard ini aktif, tekanan Zyara ke target itu makin besar
serangan atau skill ke target itu bisa memberi mark, bonus burst, atau duel pressure
ini cocok untuk menjatuhkan ancaman inti lawan

Pelepasan Crownfire utama:

Royal Slash
Butuh 2 Crownfire.
Zyara melepas tebasan api cepat.
Efek tambahannya mengikuti Standard aktif:
Advance = lebih agresif
Holdfast = memberi perlindungan kecil sesudah serang
Judgment = bonus damage ke target hukuman

Banner Ignition
Butuh 3 Crownfire.
Zyara menyalakan aura kerajaan singkat di sekitar tim.
Advance = tim mendapat dorongan ofensif
Holdfast = tim mendapat perlindungan
Judgment = target hukuman menjadi lebih rentan

Dragon Oath
Butuh 5 Crownfire.
Ini adalah pelepasan terbesar.
Zyara memanggil api darah kerajaannya untuk satu momen besar:
Advance = serangan garis depan yang sangat kuat
Holdfast = perlindungan besar sambil membalas
Judgment = finisher berat ke target yang dipilih

Ada keputusan penting lain:

pemain bisa mengganti Standard kapan saja
tapi pergantian itu menghabiskan 1 Crownfire
ini menjaga identitas Zyara sebagai pemimpin yang disiplin: dia bisa beradaptasi, tapi tidak tanpa biaya

Jadi keputusan pemain adalah:

mau mempertahankan Advance untuk tempo agresif
beralih ke Holdfast saat tim mulai ditekan
atau mengunci Judgment pada satu target lalu menabung Crownfire untuk hukuman besar

Intinya:
Crownfire Standard membuat Zyara terasa seperti putri-pejuang sejati. Api miliknya bukan ledakan kacau, tapi lambang kepemimpinan yang bisa berubah menjadi serangan, perlindungan, atau penghakiman sesuai apa yang dibutuhkan medan perang.
